🎮 Tic Tac Toe Game<br>

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project demonstrates basic front-end development skills including DOM manipulation, game logic, and<br> responsive design.<br>

🚀 Features<br>

Two-player mode (Player X vs Player O)<br>
Real-time game updates<br>
Win detection logic<br>
Draw detection<br>
Restart / Reset game option<br>
Simple and clean UI<br>

🛠️ Technologies Used

HTML – Structure of the game<br>
CSS – Styling and layout<br>
JavaScript – Game logic and interactivity

🎯 Game Rules<br>

The game is played on a 3x3 grid<br>
Player X goes first, followed by Player O<br>
Players take turns marking a cell<br>
The first player to get 3 in a row (horizontal, vertical, or diagonal) wins<br>
If all cells are filled without a winner, it's a draw<br>
